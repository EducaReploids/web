LO MÁS VISTO

DISEÑO WEB INFORMATIVO

Este curso/taller te enseña los fundamentos del diseño web, desde HTML y CSS hasta técnicas intermedias como el diseño responsivo. Ideal para principiantes y quienes buscan mejorar sus habilidades.

BASE DE DATOS CON EXCEL

Este curso/taller te guía en el uso de Excel desde lo básico hasta temas avanzados como el uso de Macros y la creación de bases de datos. Ideal para quienes desean dominar herramientas clave para la gestión eficiente de datos.

ANIMACION BASICA CON ANIMATE CC

Este curso/taller te enseña a usar Adobe Animate CC desde lo básico hasta técnicas avanzadas. Aprenderás a crear animaciones interactivas y contenido multimedia, ideal para quienes desean dar vida a sus ideas digitales.

CREACION DE APLICACIONES CON APPCREATOR24

Este curso/taller te guía en la creación de aplicaciones móviles con AppCreator24, desde lo básico hasta funciones avanzadas. Ideal para quienes desean desarrollar aplicaciones sin necesidad de programación.
